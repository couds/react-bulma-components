The container component is a basic layout element that centers your contents horizontally.

## Fluid container

If the `fluid` prop is set to true, the container will span the entire width of available space,
and leave 32px margin on the left and right sides.

<Canvas>
  <Story story={stories.Fluid} />
</Canvas>

## Breakpoints

You can have a **fullwidth** container until the specified breakpoint using the `breakpoint` prop.
