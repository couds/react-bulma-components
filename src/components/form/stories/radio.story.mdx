# Radio

<Canvas>
  <Story story={stories.BasicUsage} />
</Canvas>

The `<Form.Radio>` component is a simple wrapper around the `<input type="radio"`> HTML element.
The radio buttons are not styled to preserve cross browser compatibility.

<Message color="primary">
  <Message.Header>This is a controlled component</Message.Header>
  <Message.Body>
    <code>Form.Radio</code> is a <strong>controlled component</strong>. You need
    to pass in value, an <code>onChange</code> handler and whether the radio is
    checked in order to see changes.
  </Message.Body>
</Message>
