## Components

The `Dropdown` component contains the following components:

- `<Dropdown.Item />`: Defines an item in the dropdown menu.
- `<Dropdown.Divider />`: Draws a divider in the dropdown menu.

## Custom content

By default, `<Dropdown.Item />` is rendered as a `<div>`,
so you can put any content you like inside it.

## Right alignment

You set the `right` prop to true to align the dropdown to the right.

  <Story story={stories.RightAlignment} height="300px" />

## Menu above button

You can have the menu appear above the button instead of below by setting the `up` prop to true.
