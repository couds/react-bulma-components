# Tile

A single tile element to build 2-dimensional Metro-like, Pinterest-like, or whatever-you-like grids

## Nesting

You need at least 3 levels of hierarchy:

```
tile is-ancestor
|
└───tile is-parent
    |
    └───tile is-child
```

```
tile is-ancestor
|
├───tile is-vertical is-8
|   |
|   ├───tile
|   |   |
|   |   ├───tile is-parent is-vertical
|   |   |   ├───tile is-child
|   |   |   └───tile is-child
|   |   |
|   |   └───tile is-parent
|   |       └───tile is-child
|   |
|   └───tile is-parent
|       └───tile is-child
|
└───tile is-parent
    └───tile is-child
```
